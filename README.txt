# 🛒 Retail Sales Trend Analysis & Inventory Optimization

## 📌 Project Objective
This project aims to analyze retail sales data to uncover sales trends, forecast future demand, and optimize inventory levels.

## 🧰 Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Scikit-learn)
- SQL (for queries)
- Tableau / Power BI (dashboard)
- Jupyter Notebook
- Excel (initial cleanup)

## 📊 Key Analysis
- Sales trend analysis (monthly, daily, seasonal)
- Top/bottom performing products
- Store performance comparison
- Demand forecasting using Prophet
- Inventory restocking strategy
